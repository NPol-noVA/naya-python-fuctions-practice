# Student Grade Management System

## Description

This project is a simple Student Grade Management System built using Python functions.

## Features

- Add students
- Calculate the average grade
- Search for a student by name
- Display all students
- Display the student with the highest grade

## Files

- `main.py` - Runs the program.
- `functions.py` - Contains all program functions.
- `examples/` - Contains sample output.

## How to Run

1. Open a terminal.
2. Navigate to the project folder.
3. Run:

```bash
python main.py
```

## Example Output

```
--- Student Report ---
Alice - Grade: 90
Bob - Grade: 85
Charlie - Grade: 95

Highest Grade: Charlie (95)

Average Grade: 90.00

Enter a student name to search: Bob
```
